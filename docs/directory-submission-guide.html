2 Register on MCP Registry

The open MCP Registry gets Blackveil DNS visible in VS Code Copilot, Cursor, Windsurf, and other MCP clients.

Open MCP Registry
  • Sign up / log in
  • Start domain verification for blackveilsecurity.com
  • Copy the TXT record value provided
TypeNameValue
TXT _mcp-verify (paste value from registry)
Open Cloudflare Dashboard
  • Add DNS TXT record in Cloudflare
  • Wait ~5 minutes for propagation
  • Complete verification in MCP Registry

3 Initial MCP Registry Publish + GitHub Secret

First publish is manual. Subsequent publishes happen automatically via bv-mcp CI on every v* tag.

cd /Applications/Github/bv-mcp && npx mcp-publisher publish
  • Go to bv-mcp → Settings → Secrets → Actions
  • Add secret: MCP_REGISTRY_TOKEN = (token from registry account)
